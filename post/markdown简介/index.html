Markdown用法简介

Markdown是一种可以使用普通文本编辑器编写的标记语言，通过简单的标记语法，它可以使普通文本内容具有一定的格式。Markdown的语法简洁明了、学习容易，而且功能比纯文本更强，因此有很多人用它写博客。下面简单介绍下常用语法：

1.标题

标题能显示出文章的结构。行首插入1-6个 # ，每增加一个 # 表示更深入层次的内容，对应到标题的深度由 1-6 阶。如下：

- H1 :# Header 1
- H2 :## Header 2
- H3 :### Header 3
- H4 :#### Header 4
- H5 :##### Header 5
- H6 :###### Header 6

效果：

Header 1

Header 2

Header 3

Header 4

Header 5
Header 6

2.无序列表

可以用”- “来创建，如下：

- A
- B
- C
效果：
  • A
  • B
  • C

3. 有序列表

可以用数字后面加”.“来实现，如下：

1. Lester
2. Ever
3. Ann
效果：
  1. Lester
  2. Ever
  3. Ann

4. 引用

对某部分内容做一些说明或者引用，如下：

> 不以结婚为目的得谈恋爱都是耍流氓。——XXX
效果：

不以结婚为目的得谈恋爱都是耍流氓。——XXX

5. 分割线

分割线可以由 * - _ 这三个符号的至少三个符号表示，如下：

***
---
___

6. 链接

支持两种链接方式，行内式和参数式，链接文字都是用[方括号]标记，下面介绍行内式：

[这里是123导航](https://www.hao123.com/)
效果：

这里是123导航

7. 图片

用法跟链接基本一样，唯一不同就是图片前面要写一个！，如下

![图片名](图片地址链接)
效果：

8. 代码框

高逼格显示代码，一定要学，用两个```将代码框起来，如下：


import pandas as pd
import numpy as np

9. 表格

表格有点麻烦，请看代码：

Date | Index | Price 
-|-|-
2018-1-1 | CSI300 | 3200
效果：
Date Index Price
2018-1-1 CSI300 3200

10. 强调

一个星号和一个下划线包起来，会转换为斜体，如果两个，会加粗,如下：

*斜体*
**加粗**
效果：

斜体 加粗
